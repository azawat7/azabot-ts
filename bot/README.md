# Discord Bot

A Discord bot built with TypeScript, discord.js, and MongoDB

## Setup

### Prerequisites
- Node.js 16+
- MongoDB Collection

### Installation
```bash
npm install
```

### Environment Variables
Create `.env`:
```env
TOKEN=your_discord_bot_token
MONGODB_URI=your_mongodb_connection_string
GUILDID=your_guild_id
```

### Development
```bash
npm run dev
```
